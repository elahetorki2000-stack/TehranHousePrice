# TehranHousePrice
سامانه پیش‌بینی قیمت مسکن تهران (پروژه دِمیس). معماری مجزا (Decoupled) با بک‌اند مستقل Django و کلاینت Tkinter. آموزش مدل یادگیری ماشین روی دیتابیس واقعی مسکن. رابط گرافیکی کاملاً فارسی، راست‌چین و بهینه‌سازی شده. منوی کشویی الفبایی با پوشش ۱۰۰٪ تمام محله‌های دیتابیس. ارتباط لایه‌ها با JSON و نمایش قیمت به تومان.
